## Let's Run it!!
1. Clone the repo or download zip.
2. Navigate to the cloned directory or to the extracted directory.
3. Open terminal in the directory.
4. Run the following commands:
5. `python -m venv .env`
6. `.env\Scripts\activate.ps1` (for powershell)
7. `python main.py`
8. `deactivate`
